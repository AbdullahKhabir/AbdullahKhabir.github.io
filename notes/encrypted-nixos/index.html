# Why NixOS

Tired of other distributions there. Just want to try something new. Been using it a few years back. But not that deep.

# Download

Head over to Downloads page and get the latest iso file. I chose the minimal install iso.

Insert a USB drive to your machine and write the NixOS image to it.

dd if=nixos.iso of=/dev/sdX bs=4M status=progress

Boot your machine and interrupt it by pressing the dedicated key. Boot from USB drive.

# Connect to internet

Become root by invoking sudo -i and connect to internet using wpa_supplicant.

wpa_passphrase 'your network name' 'your passphrase' > /etc/wpa_supplicant.conf
systemctl start wpa_supplicant

or you can use wpa_cli to connect.

Type p to check if everything is fine. And once satisfied, type w and hit enter to save the work.

# Create encrypted container

Now create a LUKS container.

cryptsetup luksFormat /dev/sda2

It will ask you to confirm. Type YES in all uppercase. Then choose a good passphrase. Write it if you can’t memorize it. Once forgotten, you can’t recover it.

Now open the new container.

cryptsetup open /dev/sda2 cryptlvm

Create a physical volume in this LUKS container.

pvcreate /dev/mapper/cryptlvm

Create a volume group. I’ll name it nix.

vgcreate nix /dev/mapper/cryptlvm

Create logical volumes on volume group. I’m creating three, one for encrypted swap and another for home and last one for root.

lvcreate -L 8G nix -n swap
lvcreate -L 25G nix -n root
lvcreate -l +100%FREE nix -n home

# Formating Drive

Next we are going to create filesystems on it.

First partition will have FAT filesystem and we will add a label nixos-boot to it.

mkfs.fat -F 32 -n nix-boot /dev/sda1

Next we will create ext4 filesystems for root and home. And add swap.

mkfs.ext4 /dev/mapper/nix-root -L nix-root
mkfs.ext4 /dev/mapper/nix-home -L nix-home
mkswap /dev/mapper/nix-swap -L nix-swap

# Mount the partitions

Next we will mount the partitions. Order is important. First of all, we will mount the root, then create directories and mount the other filesystems.

mount /dev/disk/by-label/nix-root /mnt
mkdir -p /mnt/home
mkdir -p /mnt/boot
mount /dev/disk/by-label/nix-home /mnt/home
mount /dev/disk/by-label/nix-boot /mnt/boot
swapon /dev/disk/by-label/nix-swap

# Generate NixOS configuration

Next we have to generate NixOS configuration file. All OS is based on this configuration file.

nixos-generate-config --root /mnt

Let’s edit this file to have system like how we want. If you’re installing using minimal iso file, you should have nano only. In that case you can install vim .


# In case you don't have vim

nix-env -f '<nixpkgs>' -iA vim

And edit the configuration file.

vi /mnt/etc/nixos/configuration.nix

Add a user and don’t forget to enable systemd-boot else your system won’t boot.

# User management

    users = {
        users.ak = {
            isNormalUser = true;
            shell = pkgs.zsh;
            extraGroups = [
            "wheel" "ak" "docker" "video" "audio"
            ];
        }
    }

Replace ak with your desired username. Don’t forget to set a password.

# Install

Let’s install the OS now.

nixos-install

It witll install the OS based on your configuration. At the last step, it will ask you the root password. Just set the root password and reboot.


setting root password...
New password: ***
Retype new password: ***

reboot

# Login

On the first boot, login with root user, change the password, and if you previously didn’t add a user, add now.


useradd -c 'Abdullah Khabir' -m -U -G wheel,audio,video ak
passwd ak

Replace ak with your desired username, Abdullah Khabir with your real name. -U will create a group with the same name as your user, -m will create a home directory and with -G you will be added to wheel group to use sudo command, video group to set/change brightness.
